folder structure 
all components goes in the  src/componets folder
all pages goes in the src/pages folder
all svg/icons goes in the src/assets/icons folder
all related api calls should have there own file example auth realted api calls src/api/auth.ts
all api calls should use axios instance export form src/api/index.ts
all components in the routes file must be lazy loaded 
always use options api unless you are just importing modules 
use mixins if you see that a component my share the same functionality
